Scoretracer

A web scrapping project that scraps data from cricinfo.com to display the details of the latest matches ongoing

View project on GitHub

Welcome to SCORE TRACER's Page.

The web scrapping project scraps data from cricinfo.com. It lists the matches that are currently ongoing for the user. If the user selects a match then it displays the details about the match. It displays the live scores for the user. The scores gets updates.
